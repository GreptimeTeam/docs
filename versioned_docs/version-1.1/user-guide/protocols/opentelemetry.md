# OpenTelemetry (OTLP)

GreptimeDB is an observability backend to consume OpenTelemetry metrics, logs, and traces natively via [OTLP/HTTP](https://opentelemetry.io/docs/specs/otlp/#otlphttp) protocol.
Please refer to the [Ingest Data with OpenTelemetry](/user-guide/ingest-data/for-observability/opentelemetry.md) document for detailed information.
