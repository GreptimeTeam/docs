# Supported Protocols

GreptimeDB supports the following interacting protocols:

- [Greptime HTTP API][1]
- [Greptime gRPC][2]
- [MySQL protocol plugin][3]
- [PostgreSQL protocol plugin][4]
- [OpenTSDB  write protocol][5]
- [InfluxDB line protocol][6]
- [Prometheus Remote Endpoints][7]
