# Vibe Coding

## GreptimeDB MCP Server

The [GreptimeDB MCP Server](../user-guide/integrations/mcp.md) provides a Model
Context Protocol implementation that enables AI assistants like Claude to
securely explore and analyze your GreptimeDB databases.

## GreptimeDB Skills

GreptimeDB is compatible with a few open standards and protocols, that is
already learned by most coding agent. But for some of our own features like
pipeline and triggers, it would be helpful to include these skills to you coding
agent for best performance.

See the [skills](https://github.com/GreptimeTeam/docs/tree/main/skills) repo
page for more details.
