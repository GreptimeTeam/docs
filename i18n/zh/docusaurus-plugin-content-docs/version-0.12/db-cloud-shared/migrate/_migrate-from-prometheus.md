GreptimeDB 可以用来存储 [Prometheus](https://prometheus.io/) 的时间序列数据。
此外，GreptimeDB 通过其 HTTP API 支持 Prometheus 查询语言。
这可以使你轻松将 Prometheus 的 long-term storage 切换到 GreptimeDB。
