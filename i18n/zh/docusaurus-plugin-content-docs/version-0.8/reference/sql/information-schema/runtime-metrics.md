# RUNTIME_METRICS

`RUNTIME_METRICS`表提供系统运行时指标。它包括集群中`/metrics` HTTP端点输出的所有指标。

```sql
USE INFORMATION_SCHEMA;

SELECT * FROM RUNTIME_METRICS;
```

结果中的列：

* `metric_name`：指标名称。
* `value`：指标值。
* `labels`：指标标签和值，用`,`分隔。
* `node:` 指标来自哪个节点
* `node_type`：节点类型，如`datanode`、`frontend`等。
* `timestamp`：指标时间戳
