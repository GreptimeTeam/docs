# Vibe Coding

## GreptimeDB MCP Server

[GreptimeDB MCP Server](../user-guide/integrations/mcp.md) 提供了模型上下文协议
的实现，使 Claude 等 AI 助手能够安全地探索和分析您的 GreptimeDB 数据库。

## GreptimeDB Skills

GreptimeDB 兼容许多开放协议，这些协议已经被主要大语言模型所熟悉。而针对我们都有的
一些功能，可以使用 Skills 来提升 coding agent 的使用效果。

前往 [skills](https://github.com/GreptimeTeam/docs/tree/main/skills) 页面了解更
多信息，并安装这些 Skills。
