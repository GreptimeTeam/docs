# Java

GreptimeDB Java Ingester 是一个轻量级、高性能的客户端，专为向 GreptimeDB 高效写入时间序列数据而设计。基于 gRPC 协议构建，提供非阻塞、纯异步的 API，在保持与应用程序无缝集成的同时提供高吞吐写入数据的能力。

## 主要特性

- **高性能**：非阻塞、纯异步 API
- **灵活写入**：支持常规写入和批量流式（Bulk API）写入
- **生产就绪**：内置指标收集和调试功能
- **易于集成**：轻量级库，依赖最少
